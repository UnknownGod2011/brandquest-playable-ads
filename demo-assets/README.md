# Demo Custom Game Metadata

This folder contains safe metadata for the submission video.

Use `spotify-vibe-rush-custom-game.json` by copying its contents into the Custom Games form's "Import metadata JSON" box, then click "Import JSON". The form only parses metadata fields and never executes uploaded code.

After creator submission, an admin can approve the request. Approval maps the metadata to BrandQuest's trusted first-party `Brand Rush Runner` runtime, so the approved campaign becomes playable without running arbitrary uploaded JavaScript.
